# Demo screenshot / GIF

1. Start the backend (`python app.py` in `backend/`) and open `frontend/index.html`.
2. Click **Aspirin** (or **Analyze**) so the gauge and heatmap fill in.
3. Optionally click **Download PDF** to show export.
4. Record a 10–20s clip or take a PNG.

Save the file here as **`demo.gif`** or **`demo.png`** and link it from your main **`README.md`**:

```markdown
![ToxScan demo](docs/demo.gif)
```
